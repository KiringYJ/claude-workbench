# Legacy Claude Workbench Archive

This directory preserves the previous Claude Code marketplace/plugin-oriented implementation for reference.

The current architecture is vendor-neutral and uses:

- `AI_AGENT_GUIDE.md` as the canonical generated guide.
- `AI_AGENT_PROJECT.md` for project-specific manual content.
- Thin `CLAUDE.md`, `AGENTS.md`, and `GEMINI.md` entrypoints.
- Prompt-driven sync from `prompts/sync-agent-workbench.md`.

Do not use this archive as the primary distribution path. It may be useful when mining old hooks, skills, or review language for future vendor-neutral modules.
