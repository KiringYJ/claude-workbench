# Update Documentation

Review all project documentation and update anything that is stale or inaccurate.

## Checklist

1. **Read each documentation file** (README.md, CLAUDE.md, etc.)
2. **Verify file structure** — does the documented tree match the actual filesystem?
3. **Verify usage instructions** — are setup steps, config examples, and commands still accurate?
4. **Verify descriptions** — does the project summary reflect what the project actually does?
5. **Fix any inaccuracies** — edit the documentation files directly.
6. **Stage changes** — `git add` any updated documentation files.
